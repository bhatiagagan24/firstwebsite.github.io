Hosting a static page on Github

This blog will explain how to host a static page like this on Github

PS: The first image has been taken from the web

1) Open the Github and Sign Up on Github

2) Click on start a project or new Repository

3) Now fill in the repository name with your domain name along with .github.io at the end. After that set the repository as private and then click on Create Repository

4) Now upload the CSS and HTML files you want to upload but make sure that your HTML file be named index.html and after this commit the changes.

4) Now click on settings and change the settings that you want.

5) If you have followed all these steps correctly you will see the url of your website.

Now go and create your own website. HAPPY LEARNING !!
